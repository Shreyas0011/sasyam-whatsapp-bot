Sasyam Automation is a WhatsApp-based customer interaction system built using the Meta WhatsApp Business API and Yellow.ai to automate user communication. The project was designed to reduce manual support workload and provide users with faster responses through structured conversational workflows.

The system processes incoming WhatsApp messages through automated conversation flows that can respond to common queries, guide users through services, and escalate complex issues to a live agent when required. It also supports ticket creation workflows for tracking user issues.

Key Features

Automated conversational workflows

Real-time WhatsApp message handling

Live agent handover for unresolved queries

Support ticket generation

Tech Stack: Meta WhatsApp Business API, Yellow.ai, Conversational AI workflows
